使用Ubuntu + hexo 部署博客

前言: 之前在windows 10 使用过git+hexo, 安装过程还算容易, 因为node.js 有exe 安装包, 而在Ubuntu平台上, 需要利用nvm,npm包管理工具来安装, 因为没有在网上搜索到详细的解决方案, 所以主要在这里浪费了时间.

####安装git, 对接github, 这里比较简单, 都可以百度.

####安装node.js, 大概有两种方法, 一种是通过软件仓库安装, 另一种是通过nvm安装, 我使用的是后一种, 这里重点分享一下: 运行命令如下:

    export NVM_DIR="$HOME/.nvm" && (
    git clone https://github.com/creationix/nvm.git "$NVM_DIR"
    cd "$NVM_DIR"
    it checkout `git describe --abbrev=0 --tags --match "v[0-9]*" origin`
    ) && . "$NVM_DIR/nvm.sh"

等待完成，为了使 nvm 能开机自动启动，需要添加如下内容到你的~/.bashrc,~/.profile,或者~/.zshrc 文件中：

    export NVM_DIR="$HOME/.nvm"
    [ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh"

nvm 的部分使用说明

列出可安装包信息

nvm ls-remote

安装最新稳定版

nvm install stable

启用安装好的版本

nvm use node

你也可以这样启用它

nvm run node --version

设置默认版本

nvm alias default node

安装完成后使用nvm -ls检查node.js 运行状态

如果有哪一项没有安装, 可以使用 nvm install 来安装

####手动更新nvm

    (cd "$NVM_DIR"
    git fetch origin
    git checkout `git describe --abbrev=0 --tags --match "v[0-9]*" origin`
    ) && . "$NVM_DIR/nvm.sh"

####安装hexo 如果前面的步骤都正确没问题，那么现在可以通过 npm 安装 hexo 了：

npm install -g hexo-cli

####安装完成后，进行初始化操作

mkdir blog
hexo init blog      #初始化，创建一个你专门存放博客文件的文件夹，我这里把文件夹命名为 blog，你可以改成你想要的名字
cd blog                #进入 blog 目录
npm install          #安装相关依赖

一些使用说明

####hexo 更新

npm update -g

####安装插件

npm install <plugin-name> --save

####卸载插件

npm uninstall <plugin-name>
